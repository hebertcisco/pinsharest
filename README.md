# PinSharest

> Easily create new Pins for Pinterest;

You can fork or clone this repo, if you want clone make this:

```sh
git clone https://github.com/tecnobert/pinsharest
```

To start, run the following commands:

```sh
cd pinsharest
npm install
npm start
```
